FAQ

No one asked these question, so the owner had to think all of them...

What's the language of the bot? Library?

It's written in JavaScript, with the library discord.js.

Why doesn't the bot work?

There are some reasons that doesn't work: code errors, you gave the wrong permission, you misspelled the prefix or command, etc.

Why does the bot require Administrator permission?

It requires it since it would be easier to setup the permissions, but if you're skilled with setting up permissions, you can setup your own permissions.

Why did you coded the bot?

In September 2019, in my Discord server had some custom response setup with Dyno bot, and someone asked that why does it respond with Dyno instead of my bot, so I coded one. At first I'd just want a server bot, but now I decided to make it for everyone.

Can I self-host the bot?

Yes, of course you can, but you are not allowed to promote the bot or brand the bot as yours, such as telling someone that you made it, then I will definitely sue you.

Can I contribute to the bot?

Yes, of course you can, but please do not pull requests in GitHub, I don't host my bot there, I manually update GitHub once I made any changes to the bot, so just DM me with the code.

Does the bot hack users?

No, the bot doesn't hack Discord servers even it is called "hackerman14", the name was just for fun.
