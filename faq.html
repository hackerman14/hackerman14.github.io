FAQ

No one asked these question, so the owner had to think all of them...

What's the language of the bot? Library?

It's written in JavaScript, with the library discord.js.

Why the bot isn't working?

There are some reasons causing the bot not working: code errors (for this one, please contact the owner to fix it), you gave the wrong permission, you misspelled the prefix or command, etc.

Why does the bot require Administrator permission?

It requires it since it would be easier to setup the permissions, but if you're skilled with setting up permissions, you can setup your own permissions. (or if you're concerned if the bot isn't safe)

Why did you code the bot?

In September 2019, in my Discord server had some custom response setup with Dyno bot, and someone asked that why does it respond with Dyno instead of my bot, so I coded one. At first I'd just want a server bot, but now I decided to make it for everyone.

Why the bot's name is "hackerman14"?

If you've been following Skeppy's channel, in one of his video, he pretend to be a random user in his Discord server listening to people talking in voice chat, so he created a Discord user called "hackerman14". If you're interested about the video, please click here.

Can I self-host the bot?

Yes, of course you can, but you are not allowed to promote the bot or brand the bot as yours, such as telling someone that you made it, then I will definitely sue you. I still recommend using the official bot instead of self hosting.

Can I contribute to the bot?

Yes, of course you can, but please do not pull requests in GitHub, I don't host my bot there, I manually update GitHub once I made any changes to the bot, so just contact me with the code and please specify which part are you making changes.

Does the bot hack users?

No, the bot doesn't hack Discord servers even it is called "hackerman14", the name was just for fun. It's 100% safe.

I have questions, but they aren't listed here. Where can I ask a question and get a response?

Simply contact me to request it and I will add your questions here!
