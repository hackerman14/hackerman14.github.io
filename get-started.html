Get Started

Don't know how to setup the bot? Here's how!

hackerman14 is a bot which is still in development (well it always is because there are like a million bugs that needs to be patched)

Invite the Bot

To invite the bot, here are the steps:

  1. Click the "Invite" button at the top of the page.
  2. Select the options from the page.
  3. Select the server that you want the bot to join, and give it the preset permission that I've given. (you can skip this if you want a custom permission setup)
  4. Click the "Authorize" button.
  5. Do the reCAPTCHA.
  6. Done, close the Authorize tab and check your server

Interact with the Bot

Just simply type "rh!help" and all the commands will show up!
